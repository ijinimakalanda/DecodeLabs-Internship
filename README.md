# DecodeLabs - Project01_FrontendDevelopment

## 📖 Overview
This project demonstrates the fundamentals of **Frontend Development** using **HTML, CSS, and Semantic HTML**.  
It highlights how structure (HTML) and styling (CSS) work together to build modern, accessible, and responsive web pages.

---

## 🏗️ Technologies Used
- **HTML (HyperText Markup Language)** → Defines structure and content.
- **CSS (Cascading Style Sheets)** → Controls design, layout, and responsiveness.
- **Semantic HTML** → Improves readability, accessibility, and SEO.
